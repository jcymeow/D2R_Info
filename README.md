### 暗黑破坏神II:重制版 资料页
### Diablo II: Resurrected Info Query HTML Project

* 特色1: 本地页面,无需容器,无网络延迟
* 特色2: 符文之语页,底材支持向上兼容 eg.长矛包括(长矛+近战武器+武器) 弓/弩包括(弓/弩+武器)
* 特色3: 暗金物品页,包含升级后的需求
* 特色4: 套装装备页,包含升级后的需求

* 在线版地址 https://jcymeow.github.io/D2R_Info/index.html
* git仓库 https://github.com/jcymeow/D2R_Info
